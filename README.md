# Eurotel

Marketing website for **Eurotel** — a Portuguese-heritage hotel in Diu, India.

Single-page homepage template built to sell the experience (enquiry-led, no live booking engine):

- **Hero** — full-bleed video with a blurred backdrop, green tint, and a centered wordmark lockup
- **Story** — a scroll-pinned reveal where the text lights up word-by-word
- **Rooms** — a scroll-scrubbed showcase where the video scrubs room-to-room and holds on each
- **Menu overlay** — full-screen navigation

Palette: whitewash, heritage green, brass, with a whisper of terracotta. Type: Fraunces + Jost.

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
